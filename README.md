# 📊 Job Postings Analysis

A comprehensive analysis of job market trends using real-time job posting data from an API. This project analyzes job postings by location and technical skills, providing valuable insights into current market demand.

## 🎯 Project Overview

This Jupyter notebook analyzes job posting data to answer key questions about the job market:
- Which cities have the most job opportunities?
- What technical skills are most in demand?
- How do different locations compare in terms of job availability?

## ✨ Features

- **Location-Based Analysis**: Analyze job postings across 7 major US cities
- **Skills-Based Analysis**: Track demand for 12 different programming languages and technologies
- **Excel Report Generation**: Automatically generates formatted Excel spreadsheets with:
  - Raw data sheets
  - Sorted views by popularity
  - Professional formatting
- **Statistical Summary**: Provides key metrics and insights about the data
